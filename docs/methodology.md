# 방법론: fear-flow-v3

## 측정 대상

거래일 `t`의 KOSPI 수익률 `r_t`와 개인 순매수대금/당일 KOSPI 거래대금인 `flow_share_t`를 사용한다. 현재일을 제외한 직전 252거래일만으로 실전 후보는 Huber IRLS 강건 회귀, 감사 기준선은 OLS를 적합한다.

`flow_share_i = alpha_t + beta_t * r_i + epsilon_i`

현재 잔차는 학습 잔차의 중앙값과 MAD로 강건 표준화하고, 같은 학습 잔차에서의 경험적 백분위로 상태를 정한다. 최소 200개 완전 관측치, 음의 기울기, 0이 아닌 MAD가 필요하며 `R² < 0.20` 또는 `beta >= 0`이면 값을 표시하되 신규 거래는 만들지 않는다.

| 백분위 | 연구 상태 |
| --- | --- |
| 0~5 | 극단적 공포 |
| (5, 20] | 공포 |
| (20, 80) | 중립 |
| [80, 95) | 탐욕 |
| 95~100 | 극단적 탐욕 |

PDF 원문 근사 비교 모델은 개인 순매수대금을 1조원 단위로 OLS 회귀한다. 시장 규모 변화와 이상점에 취약하므로 기본 전략에 쓰지 않는다. 강건 회귀를 기본 후보로 둔 것은 과거 성과가 좋아서가 아니라 극단 관측이 다음 회귀선을 과도하게 당기는 영향을 사전에 줄이기 위해서다.

세 모델의 최신 상태·백분위·잔차·강건 z·회귀계수·적합도·학습 표본 수·품질·거래 가능 여부를 병렬로 공개한다. 화면에서 모형을 바꾸는 동작은 이 사전 계산 스냅샷·산점도·사건 결과만 선택하며 회귀를 다시 적합하지 않는다. 원문에 표시된 11개 날짜와 2026년 회귀는 별도 `pdfReplica`에 격리해 임계값 선택과 거래에서 제외한다.

산점도의 공포·탐욕 영역은 원문의 손그림 타원을 복제하지 않는다. 최신 회귀의 252개 학습 잔차를 정렬하고 경험적 CDF의 상태 전이 순서통계량 `q5`, `q20`, `q80`, `q95`를 계산한다. 각 경계는 `flow = alpha + beta * return + q`인 회귀선 평행선이다. 화면은 서버가 공개한 이 경계만 사용하며 과거 점의 당시 상태나 반올림된 좌표로 경계를 다시 추정하지 않는다. 최신 모형이 `unavailable`이면 영역도 표시하지 않는다.

## 사건과 전략

연속된 같은 극단 상태는 첫날만 사건으로 센다. 공급자 결측이나 `low_model_fit`은 기존 극단 상태를 리셋하지 않으므로 장애 뒤 같은 상태가 다시 나타났다고 새 사건을 만들지 않는다. 기본 사건 표본은 20거래일 창이 겹치지 않게 만들고, 전체 사건은 보조로 둔다. 1·5·10·20거래일 선행수익률마다 사건 수, 평균, 중앙값, 양의 수익률 비중과 시계열 의존성을 완화한 10,000회 이동블록 bootstrap 평균 95% 신뢰구간을 공개한다. 모든 거래일의 평균 선행수익률을 무조건부 벤치마크로 두고 평균 초과수익과 그 신뢰구간을 별도로 공개한다. 이 무조건부 평균만 전달된 경우 초과수익 신뢰구간은 벤치마크 평균 추정치를 고정한 조건부 구간이며, 벤치마크 평균 자체의 추정 오차는 포함하지 않는다. `meanExcessReturnCi95BenchmarkTreatment=fixed_external_mean`으로 이를 기계 판독 가능하게 표시한다. 사건별 벤치마크 수익률을 함께 재표집한 경우에는 `paired_event_returns`, 초과수익 구간이 없으면 `unavailable`을 사용한다. 20개 미만 표본은 소표본으로 표시한다.

기본 롱/현금 정책은 유효한 극단적 공포 최초 진입을 확인한 다음 거래일 조정시가에 226490을 매수한다. 기본 청산선은 백분위 80 회복이며, 그 신호 다음 거래일 시가 또는 진입 후 최대 20거래일 규칙에 따라 다음 시가에 청산한다. 기존 백분위 50 청산은 10bp 고정 민감도 진단으로만 남기며 결과가 좋아도 기본 규칙으로 되돌려 승격하지 않는다. 동시 포지션은 하나이고 보유 중 같은 방향의 반복 신호는 무시한다. 따라서 차트의 극단 공포 관측일 수는 실제 진입 수와 같지 않다.

롱/숏/현금 정책은 같은 롱 규칙에 더해 유효한 극단적 탐욕 최초 진입을 확인한 다음 거래일 시가에 합성 숏을 연다. 기본 숏 청산선은 백분위 20 이하 회복이며 최대 보유기간은 동일하게 20거래일이다. 보유 중 반대 극단이 나타나면 다음 시가에 기존 포지션을 청산하고 같은 시가로 반대 포지션에 진입한다. 이 반전은 청산 1측과 진입 1측, 총 2측의 거래비용을 부과한다. 일반 회복 청산에는 모형 적합도가 낮아도 유효한 백분위를 사용할 수 있지만, 신규 진입·반전에는 거래 가능 품질 게이트를 요구한다. 결측 시에는 새 거래를 만들지 않고 최대 보유기간 규칙만 계속 적용한다.

합성 숏은 진입 비용을 먼저 차감한 가용자본의 1배를 고정 수량으로 매도하고 청산까지 재조정하지 않는 회계 모형이다. 조정 총수익 OHLC를 사용하므로 분할과 분배금의 경제효과는 가격 경로에 반영되지만, 대차 가능 수량·대차료·리콜·증거금·강제청산·공매도 주문 규칙·시장충격은 반영하지 않는다. 대차료와 현금·담보 수익률은 모두 0%다. 합성 숏으로 자기자본이 0 이하가 되는 경로는 fail-closed로 공개하지 않는다. 이 결과는 실제 공매도나 역ETF의 실행성과가 아니다.

노출도는 ETF 가격과 전략 포지션을 모두 산출할 수 있는 거래일 중 롱·숏·현금 비율로 각각 공개하며, 226490 상장 전 행을 현금으로 세지 않는다. 총노출은 롱과 숏 노출의 합, 순노출은 롱에서 숏을 뺀 값이다. 편도 비용은 0·5·10·20bp 전체 격자를 공개하고 10bp를 기본으로 한다. 현금 수익률은 0%다. 226490과 069500은 각 전체기간과 공통기간을 분리하며 매수·보유 외에 전략과 동일한 진입·청산 시가를 0bp로 재현한 비교군과 변동성 일치 비교군을 함께 둔다. 50일 이격도가 과거 756거래일 하위 10%일 때만 롱 진입하는 결과는 롱/현금 강건성 변형이며 대칭적인 숏 필터가 없으므로 롱/숏 정책에는 적용하지 않는다. 신호 기준일 뒤의 프록시 가격은 현재 포지션이나 성과에 포함하지 않는다.

서버 검증 기본 결과는 롱/현금의 공개 변형과 robust 롱/숏 정책의 롱 청산 80·숏 청산 20이다. 화면의 사용자 청산 입력은 정수 50~94이며 숏 청산선은 `100 - 롱 청산선`으로 대칭 설정한다. 기본 80을 제외한 값과 robust 외 롱/숏 신호 변형은 서버가 발행한 일별 백분위·거래 가능 여부·ETF 조정가격으로 브라우저가 거래 상태 경로만 다시 계산하는 사용자 시나리오다. 브라우저는 회귀계수, 잔차, 백분위, 상태, 품질 게이트나 사건 표본을 재계산하지 않는다. 사용자 시나리오는 서버 검증 기본값과 명확히 구분해 표시하고, 성과를 보고 임계값을 고르는 최적화 근거로 쓰지 않는다.

## 시간 정렬

모든 회귀 학습창은 현재일을 제외한다. ETF 체결은 신호를 종가 후 확인한 다음 거래일 조정 시가다. ETF 조정가격 세션은 전 구간 KRX 공식 거래 세션과 대조하고, 누락일 앞뒤의 조정계수가 0.5% 이내로 안정적일 때만 공식 OHLC를 같은 스케일로 보정한다. 미해결 세션이 하나라도 있으면 해당 프록시 사건 연구와 백테스트를 공개하지 않는다. Micron과 USD/KRW는 해당 KRX 거래일과 같은 날짜의 미국 세션을 쓰지 않고, 날짜 라벨이 KRX일보다 엄격히 이전인 마지막 관측만 `merge_asof`로 정렬한다.

반도체 진단은 `MU 종가 × USD/KRW`와 SK하이닉스 종가를 각각 진단창 첫 관측=100으로 지수화한다. `MU/Hynix 비율지수`와 `MU 지수 - Hynix 지수`인 상대 스프레드를 함께 공개하며, 상대 스프레드 단위는 수익률이 아니라 지수포인트다. 삼성전자·SK하이닉스의 최신 yfinance 조정종가는 같은 `dataAsOf`의 KRX Open API 원종가와 0.5% 이내일 때만 진단을 공개한다.

## 해석 한계

- 원문에는 수익률 기간, 회귀창, 수급 범위, 회귀식, 임계값이 없다.
- 색칠된 구간과 선택된 사례는 예측력 또는 인과관계를 증명하지 않는다.
- 본 아이디어는 2026년 관측 후 제안되어 전체 과거검증이 사후적·탐색적이다.
- CPI, 컨센서스, AI Capex 뉴스는 사건 설명이지 신호 입력이 아니다.
- 무료 데이터 공급자의 정정, 생존편향, ETF 추적오차와 체결 가정이 결과를 바꿀 수 있다.
- 합성 숏 결과는 대차·증거금·공매도 규칙과 시장충격을 제외하므로 실제 실행성과보다 낙관적일 수 있다.
- 화면에서 여러 청산선을 탐색하면 다중검정·선택편향이 생기며, 선택한 값은 사전 정의 규칙의 검증 결과가 아니다.
