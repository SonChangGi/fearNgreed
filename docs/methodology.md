# 방법론: fear-flow-v1

## 측정 대상

거래일 `t`의 KOSPI 수익률 `r_t`와 개인 순매수대금/당일 KOSPI 거래대금인 `flow_share_t`를 사용한다. 현재일을 제외한 직전 252거래일만으로 OLS를 적합한다.

`flow_share_i = alpha_t + beta_t * r_i + epsilon_i`

현재 잔차는 학습 잔차의 중앙값과 MAD로 강건 표준화하고, 같은 학습 잔차에서의 경험적 백분위로 상태를 정한다. 최소 200개 완전 관측치, 음의 기울기, 0이 아닌 MAD가 필요하며 `R² < 0.20` 또는 `beta >= 0`이면 값을 표시하되 신규 거래는 만들지 않는다.

| 백분위 | 연구 상태 |
| --- | --- |
| 0~5 | 극단적 공포 |
| (5, 20] | 공포 |
| (20, 80) | 중립 |
| [80, 95) | 탐욕 |
| 95~100 | 극단적 탐욕 |

원문 충실 비교 모델은 개인 순매수대금을 1조원 단위로 회귀한다. 시장 규모 변화에 취약하므로 기본 전략에 쓰지 않는다. 두 모델의 방향이 다르면 신뢰도를 `mixed`로 낮춘다.

## 사건과 전략

연속된 같은 극단 상태는 첫날만 사건으로 센다. 공급자 결측이나 `low_model_fit`은 기존 극단 상태를 리셋하지 않으므로 장애 뒤 같은 상태가 다시 나타났다고 새 사건을 만들지 않는다. 기본 사건 표본은 20거래일 창이 겹치지 않게 만들고, 전체 사건은 보조로 둔다. 1·5·10·20거래일 선행수익률마다 사건 수, 평균, 중앙값, 양의 수익률 비중과 10,000회 bootstrap 평균 95% 신뢰구간을 공개한다. 20개 미만 표본은 소표본으로 표시한다.

전략은 유효한 극단적 공포를 확인한 다음 거래일 시가에 226490을 매수하고, 백분위 50 회복 신호 다음 시가 또는 20거래일 보유 후 다음 시가에 청산한다. 동시 포지션은 하나이며 보유 중 반복 신호는 무시한다. 편도 비용은 체결금액에 10bp씩 곱해 적용하고, 민감도는 5bp와 20bp다. 현금 수익률은 0%다. 226490과 069500은 각 전체기간과 공통기간을 분리한다. 50일 이격도가 과거 756거래일 하위 10%일 때만 진입하는 결과는 강건성 변형이며 기본 모형으로 승격하지 않는다.

## 시간 정렬

모든 회귀 학습창은 현재일을 제외한다. ETF 체결은 신호를 종가 후 확인한 다음 거래일 조정 시가다. Micron과 USD/KRW는 해당 KRX 거래일과 같은 날짜의 미국 세션을 쓰지 않고, 날짜 라벨이 KRX일보다 엄격히 이전인 마지막 관측만 `merge_asof`로 정렬한다.

## 해석 한계

- 원문에는 수익률 기간, 회귀창, 수급 범위, 회귀식, 임계값이 없다.
- 색칠된 구간과 선택된 사례는 예측력 또는 인과관계를 증명하지 않는다.
- 본 아이디어는 2026년 관측 후 제안되어 전체 과거검증이 사후적·탐색적이다.
- CPI, 컨센서스, AI Capex 뉴스는 사건 설명이지 신호 입력이 아니다.
- 무료 데이터 공급자의 정정, 생존편향, ETF 추적오차와 체결 가정이 결과를 바꿀 수 있다.
